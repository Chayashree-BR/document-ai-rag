# Document AI RAG

This project is a Retrieval-Augmented Generation (RAG) system that lets users upload documents and ask questions from them.

## Features

- Upload documents
- Chunk + embed text
- Semantic search (vector DB)
- LLM answers using context

## Tech Used

Python, FAISS/Chroma, LLM API, Streamlit/Flask

## Run Project

```bash
pip install -r requirements.txt
python app.py
